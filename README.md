# 💎 Pearls Mehndi Studio

A modern, responsive frontend website for a professional mehndi booking service.  
Designed to provide a clean, elegant, and premium user experience for bridal and event mehndi services.

---

## 🌸 About the Project

**Pearls Mehndi Studio** is a beautifully designed React-based website created for showcasing mehndi services, gallery work, and enabling customers to send booking requests.

This project focuses on:

- Clean UI/UX design
- Brand consistency
- Modern layout structure
- Responsive design
- Professional section-based website flow

---

## ✨ Features

✔ Elegant Hero Section  
✔ Services Section with Hover Effects  
✔ Gallery Section with Image Grid  
✔ Why Choose Us Section  
✔ Contact Page with Form & Info Cards  
✔ Fixed Navigation Bar with Active Link Highlight  
✔ Smooth Scrolling  
✔ Themed Color System (Primary / Secondary Variables)  
✔ Responsive Layout  
✔ Professional Footer  

---

## 🎨 Theme Design

The website uses a custom theme system with CSS variables:

- Primary Color – Deep Mehndi Maroon  
- Highlight Section Color – Warm Gold Brown (#B17F59)  
- Accent Color – Soft Gold  
- Background – Light Cream Tone  

This creates a premium bridal aesthetic.

## 🚀 How to Run the Project

1. Clone the repository

```
git clone https://github.com/yourusername/pearls-mehndi-studio.git
```

2. Navigate to the project folder

```
cd pearls-mehndi-studio
```

3. Install dependencies

```
npm install
```

4. Start development server

```
npm start
```

The app will run at:

```
http://localhost:3000
```
